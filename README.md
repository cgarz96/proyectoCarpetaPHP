# proyectoCarpetaPHP
Este pequeño proyecto desarrollado en PHP (Nativo) sirve para generar carpetas, en base al nombre asignada a la misma, para luego mostrarla haciendo uso del explorador de Windows
##  Herramientas:
### ✅ PHP
### ✅ jQuery
### ✅ Bootstrap
